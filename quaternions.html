Quaternions & Hypercomplex Numbers

● ● ● ○

Beyond complex numbers — the four-dimensional algebra that rotates the world

Algebra · Advanced

In 1843, William Rowan Hamilton carved i² = j² = k² = ijk = −1 into Brougham Bridge in Dublin, creating the quaternions — a four-dimensional number system that sacrifices commutativity but gains the power to represent 3D rotations without singularities. Today quaternions are essential in computer graphics, robotics, spacecraft navigation, and anywhere smooth rotation matters.

Why it matters: This is not just a mathematical curiosity. The Apollo 11 mission nearly suffered gimbal lock. Modern spacecraft, game engines, and robotics all use quaternions to avoid this singularity.

4. The Number Systems — ℝ → ℂ → ℍ → 𝕆

The quaternions sit in a chain of number systems, each built from the last via the Cayley-Dickson construction. At each step you gain dimensions but lose an algebraic property.

ℝ — The Real Numbers (1D)

The familiar number line. Every real number is its own conjugate.
✓ Ordered ✓ Commutative ✓ Associative ✓ Division algebra

ℂ — The Complex Numbers (2D)

z = a + bi where i² = −1. Fundamental theorem of algebra: every polynomial has a root.
✓ Commutative ✓ Associative ✓ Division algebra
✗ No longer ordered (you cannot say i > 0 or i < 0)

ℍ — The Quaternions (4D)

q = a + bi + cj + dk. Three imaginary units: i² = j² = k² = ijk = −1.
✓ Associative ✓ Division algebra
✗ No longer commutative (ij = k but ji = −k)

𝕆 — The Octonions (8D)

Eight-dimensional. Seven imaginary units. Used in string theory and exceptional Lie groups.
✓ Division algebra (the last one!)
✗ No longer associative ((e1e2)e4 ≠ e1(e2e4))

Interactive: Non-commutativity

i × j = +k   but   j × i = −k
The Cayley-Dickson pattern: At each step, you take pairs of the previous number system and define a new multiplication rule. Hurwitz's theorem (1898) says only ℝ, ℂ, ℍ, and 𝕆 are normed division algebras — there is no 16-dimensional version with division. The chain stops at eight dimensions.

5. SLERP — Smooth Interpolation

Given two orientations represented as unit quaternions q&sub0; and q&sub1;, how do you smoothly transition between them? Linear interpolation (LERP) doesn't work well on the sphere. Spherical Linear Interpolation (SLERP) traces a great arc at constant angular velocity.

Applications: SLERP is used everywhere smooth rotation matters — 3D animation (interpolating keyframes), robotics (smooth joint trajectories), and spacecraft attitude control (reorienting without sudden jerks). It produces the shortest path on the rotation sphere at constant angular speed.

Where You'll See This

Quaternions are the standard for 3D rotation in video games, robotics, and aerospace. Unlike Euler angles, they don’t suffer from gimbal lock — a problem that nearly doomed the Apollo 11 mission. Every smartphone uses quaternions to track its orientation, and every 3D game engine (Unity, Unreal) uses them internally for smooth rotations.

Try This

Quaternions extend complex numbers: i² = j² = k² = ijk = −1. Verify that ij = k but ji = −k. What does this tell you about quaternion multiplication?

From ijk = −1 and i² = −1: multiply both sides of ijk = −1 on the left by i to get i²jk = −i, so −jk = −i, meaning jk = i. Similarly, from ij = k, multiply on the right by j: ijj = kj, so i(−1) = kj, meaning kj = −i. Quaternion multiplication is NOT commutative — order matters, just like matrix multiplication.
