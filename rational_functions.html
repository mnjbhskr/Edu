Rational Functions & Asymptotes

●●●● Intermediate

Understanding poles, asymptotes, and the behaviour of polynomial ratios

1. What Is a Rational Function?

A rational function is any function that can be written as the ratio of two polynomials:

f(x) = P(x) / Q(x)

where P(x) and Q(x) are polynomials and Q(x) is not identically zero. The domain of a rational function excludes every value of x where Q(x) = 0, since division by zero is undefined.

Common Examples

Simplest: f(x) = 1 / x  —  domain: x ≠ 0
Linear/linear: f(x) = (x + 1) / (x − 2)  —  domain: x ≠ 2
Quadratic/quadratic: f(x) = (x² − 1) / (x² + 1)  —  domain: all real x
Higher degree: f(x) = (x² + 3) / (x³ − x)  —  domain: x ≠ 0, ±1
Every polynomial is itself a rational function — just write it as P(x)/1. Rational functions are to polynomials what fractions are to whole numbers: a richer class that allows division.

2. Vertical Asymptotes

A vertical asymptote occurs at x = a when Q(a) = 0 but P(a) ≠ 0. Near such a point the function grows without bound:

As x → a,   |f(x)| → ∞

To find vertical asymptotes: factorise Q(x) completely, then check each zero. If the same factor appears in both P(x) and Q(x), it produces a hole (removable discontinuity) rather than an asymptote.

Holes vs Asymptotes

Example: f(x) = (x² − 4) / (x − 2) = (x − 2)(x + 2) / (x − 2)
Cancel: f(x) = x + 2   for x ≠ 2
Result: Hole at x = 2, not a vertical asymptote
Compare: g(x) = 1 / (x − 2)
Result: Vertical asymptote at x = 2 (no cancellation possible)
When approaching a vertical asymptote, the function can tend to +∞ from one side and −∞ from the other. Checking the sign of f(x) on each side reveals whether the curve rises or falls as it approaches the asymptote.

3. Horizontal & Oblique Asymptotes

The end behaviour of a rational function — what happens as x → ±∞ — depends on the degrees of P(x) and Q(x). Let n = deg(P) and m = deg(Q).

The Three Cases

Case 1: n < m  —  Horizontal asymptote at y = 0
Example: f(x) = 1/(x² + 1) → 0 as x → ±∞
Case 2: n = m  —  Horizontal asymptote at y = an/bm
Example: f(x) = (3x² + 1)/(2x² − 5) → 3/2 as x → ±∞
Case 3: n = m + 1  —  Oblique (slant) asymptote
Example: f(x) = (x² + 1)/x → the line y = x
▶ Deriving the oblique asymptote by polynomial long division

When deg(P) = deg(Q) + 1, divide P(x) by Q(x):

P(x) / Q(x) = L(x) + R(x) / Q(x)

where L(x) is linear (the quotient) and deg(R) < deg(Q). As x → ±∞, the remainder term R(x)/Q(x) → 0, so the function approaches the line y = L(x).

Worked example: f(x) = (2x² + 3x − 1) / (x + 1)

(2x² + 3x − 1) ÷ (x + 1) = 2x + 1 remainder −2
So f(x) = 2x + 1 + (−2)/(x + 1)
Oblique asymptote: y = 2x + 1
If deg(P) exceeds deg(Q) by 2 or more, the function has no linear asymptote — it grows like a polynomial. Such cases are sometimes called curvilinear asymptotes and are less commonly examined at A-level.

5. Curve Sketching Strategy

A systematic approach to sketching any rational function by hand:

Step 1 — Domain: Find where Q(x) = 0. Exclude those x-values.
Step 2 — Intercepts: y-intercept: evaluate f(0). x-intercepts: solve P(x) = 0 (where Q(x) ≠ 0).
Step 3 — Vertical asymptotes: Zeros of Q(x) that do not cancel with P(x).
Step 4 — Holes: Common factors of P(x) and Q(x) — cancel them and evaluate the simplified function at that x.
Step 5 — Horizontal / oblique: Compare degrees. Apply the three cases from Section 3.
Step 6 — Sign analysis: Test a point in each interval between vertical asymptotes and x-intercepts.
Step 7 — Behaviour near asymptotes: Check the sign of f(x) as x approaches each vertical asymptote from both sides.
Step 8 — Sketch: Plot intercepts, draw asymptotes as dashed lines, then connect with smooth curves honouring the sign analysis.

Worked Example

Sketch f(x) = (x + 1) / (x² − 4).

Domain: x ≠ ±2   (since x² − 4 = (x − 2)(x + 2))
y-intercept: f(0) = 1/(−4) = −0.25
x-intercept: x + 1 = 0 ⇒ x = −1
Vertical asymptotes: x = 2 and x = −2
Horizontal asymptote: deg(P) = 1 < deg(Q) = 2 ⇒ y = 0
Sign near x = 2: f(1.9) ≈ −7.4 (below), f(2.1) ≈ 7.6 (above)
Sign near x = −2: f(−2.1) ≈ 2.7 (above), f(−1.9) ≈ −2.3 (below)

6. Worked Examples

Example 1 (A-level)

The curve C has equation y = (2x² − 5x − 3) / (x − 3). Find the equations of all asymptotes and determine the coordinates of any holes.

▶ Show full solution

Step 1: Factorise the numerator.

2x² − 5x − 3 = (2x + 1)(x − 3)

Step 2: Simplify.

y = (2x + 1)(x − 3) / (x − 3) = 2x + 1   for x ≠ 3

Step 3: Identify features.

The factor (x − 3) cancels completely. Therefore there is no vertical asymptote — instead there is a hole at x = 3.

At the hole: y = 2(3) + 1 = 7. So the hole is at (3, 7).

The simplified function y = 2x + 1 is a straight line, so there is no horizontal or oblique asymptote in the traditional sense — the graph is the line y = 2x + 1 with a single point removed.

Example 2 (A-level)

For the function f(x) = (3x − 1) / (x² − x − 6), find all asymptotes, intercepts, and sketch the curve.

▶ Show full solution

Step 1: Factorise the denominator.

x² − x − 6 = (x − 3)(x + 2)

Step 2: Check for common factors. P(x) = 3x − 1 has no factor in common with Q(x), so there are no holes.

Step 3: Vertical asymptotes.

Q(x) = 0 when x = 3 or x = −2. Both give non-zero numerator values:

P(3) = 8 ≠ 0   ⇒   x = 3 is a vertical asymptote
P(−2) = −7 ≠ 0   ⇒   x = −2 is a vertical asymptote

Step 4: Horizontal asymptote. deg(P) = 1 < deg(Q) = 2, so y = 0.

Step 5: Intercepts.

y-intercept: f(0) = −1/(−6) = 1/6
x-intercept: 3x − 1 = 0 ⇒ x = 1/3

Step 6: Sign analysis near asymptotes.

x → 3−:   f(x) → −∞
x → 3+:   f(x) → +∞
x → −2−:   f(x) → +∞
x → −2+:   f(x) → −∞
