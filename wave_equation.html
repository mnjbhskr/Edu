The Wave Equation

●●●● Intermediate

How disturbances propagate — the PDE governing strings, sound, and light

Stretch a string and pluck it. The disturbance you create travels outward in both directions at a speed determined by tension and mass density. This behaviour is encoded in a single partial differential equation: ∂²y/∂t² = v² ∂²y/∂x². The same equation describes sound waves in air, electromagnetic waves in a vacuum, and seismic waves through the Earth — the mathematics of vibration is universal.

Wave speed v = √(T/μ) is determined by the medium (tension T, mass density μ), not by the wave itself. Two waves pass through each other unchanged — superposition is exact because the wave equation is linear.

When you drop a stone into still water, the expanding ripple obeys the wave equation in two dimensions. The speed of the ripple is determined by the water’s surface tension and depth — not by how hard you threw the stone.

The wave equation is Newton’s 2nd law applied to each infinitesimal string element. Where curvature is greatest (string most bent), acceleration is greatest. The ratio curvature-to-acceleration is always exactly v² — you can read wave speed from the colour-arrow correspondence.

Fixing the endpoints (y=0 at x=0 and x=L) quantises the allowed wavelengths: only those that fit a whole number of half-wavelengths in the string survive. This is precisely analogous to quantum mechanics, where boundary conditions on the Schrödinger wave function produce discrete energy levels. Musical instruments exploit these harmonic modes — each string or tube selects its own set.

Any shape of vibrating string can be written as a sum of these harmonic modes — this is Fourier decomposition. A guitar string plucked at its centre excites mostly odd harmonics (n = 1, 3, 5, …), which is why it sounds different from a string plucked near one end. See Fourier Transform for the full mathematical framework.
The Mathematics of Vibrating Strings

Consider a small element of string at position x with tension T and linear mass density μ. Newton’s second law applied to the transverse displacement gives μ·∂²y/∂t² = T·∂²y/∂x², which rearranges to the wave equation ∂²y/∂t² = v²∂²y/∂x² where v = √(T/μ). This derivation assumes small displacements so that the string angle stays small — the linearisation that makes superposition exact.

The general solution was found by d’Alembert in 1747: y(x,t) = f(x−vt) + g(x+vt) for any twice-differentiable functions f and g. Any shape whatsoever can propagate rightward or leftward at speed v. The two terms are completely independent and simply add by superposition. This is why a guitar string plucked in the centre sends two identical pulse shapes racing outward in opposite directions.

When the string is fixed at both ends the boundary conditions force f and g to conspire into standing waves: yn = A sin(nπx/L) cos(ωnt). These sinusoidal modes are the eigenfunctions of the wave operator on the interval [0,L], and any plucked shape decomposes into a sum of them — this is exactly Fourier’s insight. See the Fourier Transform page for how arbitrary waveforms decompose into harmonics.

In 1865 Maxwell showed that the electric and magnetic field components each satisfy the wave equation with v = 1/√(ε0μ0) = c, the speed of light. This single calculation unified electricity, magnetism, and optics. For the full Maxwell framework see Wave Interference and the companion page on Simple Harmonic Motion, which provides the single-oscillator foundation from which the full wave equation grows.

Try This

A guitar string is 0.65m long. If the wave speed on the string is 300 m/s, what is the fundamental frequency? What are the first three harmonics?

Fundamental: f₁ = v/(2L) = 300/(2×0.65) = 230.8 Hz (approximately B♭3). Harmonics: f₂ = 461.5 Hz (B♭4, one octave up), f₃ = 692.3 Hz (F5, an octave and a fifth up). The wave equation predicts exactly these harmonics — the string can only vibrate at integer multiples of the fundamental.
