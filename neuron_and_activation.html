The Single Neuron

One weighted sum, one decision — the atom from which all intelligence is built

Every neural network — from a simple spam filter to GPT-4 — is made of the same fundamental unit: the artificial neuron. It takes inputs, multiplies each by a learned weight, adds them up, applies a non-linear function, and produces an output. This page builds that unit from scratch.

2. Activation Functions — Why Non-linearity Matters

Four activation functions compared

Click a function name to highlight it. The dashed line shows its derivative.

ReLU
Sigmoid

GELU
ReLU(z) = max(0, z) — piecewise linear, most common
Sigmoid(z) = 1 / (1 + e−z) — squashes to (0, 1)
Tanh(z) = (ez − e−z) / (ez + e−z) — squashes to (−1, 1)
GELU(z) ≈ 0.5z(1 + tanh[√(2/π)(z + 0.044715z³)]) — used in GPT
Without activation functions, a million-layer network collapses to a single matrix multiplication. Non-linearity allows neural networks to approximate any function — this is the Universal Approximation Theorem.

Activation functions introduce non-linearity — the ingredient that gives depth its power.

The equation w·x + b = 0 defines a hyperplane in input space.
In 2D:  w1x1 + w2x2 + b = 0  →  a line
In 3D:  w1x1 + w2x2 + w3x3 + b = 0  →  a plane
In nD:  w·x + b = 0  →  a hyperplane
The weight vector w is perpendicular to this boundary.
The bias b shifts it away from the origin.

A single neuron partitions its input space into two half-spaces separated by a hyperplane. The weight vector determines the orientation; the bias determines the position. This is why one neuron can only solve linearly separable problems.

One neuron = one straight line. The XOR problem requires a curved boundary — that requires multiple layers. This is why deep networks exist. Minsky & Papert (1969) proved this limitation, causing the first "AI winter."

A single neuron can only separate classes with a straight line — more complex boundaries demand depth.

Why This Matters

McCulloch & Pitts (1943) showed that a simple threshold unit could compute logic. Rosenblatt (1958) built the Perceptron — the first trainable neuron. Glorot et al. (2011) showed that careful activation function choice (ReLU over sigmoid) was key to training deep networks. Every modern AI system — from image classifiers to large language models — is built by stacking and connecting billions of units identical to the one you just explored.

A single neuron draws a line. Two layers of neurons can approximate any continuous function. The depth of modern networks (hundreds of layers, trillions of parameters) is what transforms this simple unit into something that can write poetry, prove theorems, and fold proteins.

To make this neuron learn, we need to adjust its weights automatically. That is exactly what gradient descent does — it computes how much the error changes with respect to each weight, then nudges each weight in the direction that reduces the error. A single neuron trained by gradient descent on labelled data is called a Perceptron — the original neural network (Rosenblatt, 1958).
