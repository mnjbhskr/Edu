Modelling the changing variance of financial returns — quiet periods and storms

4. QQ-Plot: Raw vs Standardised Returns

Raw returns show heavy tails (points deviate from the diagonal in a QQ-plot against normal). GARCH-standardised returns (rt/σt) should be closer to normal, though some excess kurtosis typically remains.

Key GARCH stylised facts
Volatility clustering: Large returns (of either sign) tend to be followed by large returns, and small by small. This is captured by significant ACF in squared returns.
Fat tails: Financial returns have excess kurtosis (kurtosis > 3). GARCH generates this endogenously even with normal innovations.
Persistence: α + β close to 1 means volatility shocks decay slowly. Many financial series have α + β > 0.98.
Leverage effect: Negative returns tend to increase volatility more than positive returns of the same magnitude. This requires asymmetric models like GJR-GARCH or EGARCH.
Mean reversion: Despite persistence, GARCH(1,1) with α + β < 1 implies mean-reverting variance with unconditional level ω/(1 − α − β).

Monte Carlo: Persistence & Half-Life

For different persistence levels (α + β), simulate 500 GARCH paths and measure the average half-life of variance shocks. Higher persistence = longer half-life.

Try This

Set α = 0.05, β = 0.94 (typical for daily equity returns). What is the unconditional variance? Now set α = 0.15, β = 0.83. How does the QQ-plot change?

Higher α means larger ARCH effects: recent shocks have more impact on conditional variance. This leads to more volatile variance dynamics and heavier tails. The QQ-plot will show more deviation from normality in the raw returns, but GARCH-standardised returns should still be closer to the diagonal.
