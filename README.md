# MathsEdu.org — A Visual Discovery of Mathematics

**An interactive journey through 2,500 years of human thought**

[mathsedu.org](https://mathsedu.org)

---

Mathematics is the foundation of all science, yet too often it is taught as rules to memorise rather than ideas to discover. MathsEdu is a growing collection of **190+ interactive visualisations** spanning pure mathematics, physics, finance, chemistry, biology, economics, and artificial intelligence — each one a self-contained exploration designed to build genuine intuition.

## At a Glance

- **19 chapters** covering topics from Logic & Proof to Game Theory
- **14 narrative journeys** — multi-act stories that weave mathematics into real-world contexts (propulsion, flight, cryptography, chaos, and more)
- **Three ways in**: browse by level, by topic, or by journey
- **Mathematicians gallery** with 90+ biographical entries and an interactive hover-card system
- **Interactive concept map** — a navigable tree with Pure Mathematics as the trunk and applied branches

## Technology

Pure HTML, CSS, and JavaScript — no frameworks, no build process, no dependencies. Every page is a single self-contained file using the Canvas API for interactive graphics. Designed to load instantly and work on any device.

Hosted on GitHub Pages at [mathsedu.org](https://mathsedu.org).
