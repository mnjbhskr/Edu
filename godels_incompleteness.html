Gödel’s Incompleteness Theorems

Why mathematics cannot prove everything about itself — the limits of formal systems

Try This

Consider the sentence “This statement is not provable.” If the system proves it, the system has proved something false (it just proved it, so it IS provable) — making the system inconsistent. If the system cannot prove it, then the statement is true but unprovable — making the system incomplete. Which would you rather have?

This is essentially Gödel’s construction. He encoded this self-referential sentence in arithmetic using Gödel numbering. The key insight is that any sufficiently powerful system can “talk about itself” — and once it can, it faces this dilemma. Most mathematicians choose consistency over completeness.

The Self-Reference Trick

Gödel’s diagonal argument: imagine a grid where rows are statements and columns are proofs. A “✓” at (i, j) means proof j proves statement i. By flipping the diagonal, we construct a statement that no proof can reach.

6
Each row represents a statement Si and each column a potential proof Pj. The diagonal entries tell us: “Does proof j prove statement j?” Now construct a new statement G that disagrees with every diagonal entry. G cannot appear as any row — it is unprovable.

The First Incompleteness Theorem

If a formal system is consistent and sufficiently powerful (can express basic arithmetic), there exist true statements it cannot prove. Follow the logical chain below.

1 Assume our system F is consistent (never proves both P and ¬P) and can express basic arithmetic.
2 Using Gödel numbering, every formula and proof in F gets a unique number. F can “talk about” its own formulas via these numbers.
3 Construct the Gödel sentence G: “This statement (with Gödel number g) is not provable in F.”
4 Suppose G is provable in F. Then F proves “G is not provable in F” — but G is provable, so F proves something false. Contradiction! F would be inconsistent.
5 Suppose ¬G is provable in F. Then F proves “G is provable in F.” But G is not provable (we just showed that). So F proves something false. Contradiction!
6 Conclusion: Neither G nor ¬G is provable in F. The statement G is undecidable — true but unprovable.

The Second Incompleteness Theorem

No consistent system sufficiently powerful to express arithmetic can prove its own consistency. The system cannot verify itself — like a mirror trying to see behind itself.

If F could prove “F is consistent,” we could derive G (the Gödel sentence) within F — but G is unprovable in F. So F cannot prove its own consistency. You always need a stronger system to verify a weaker one.
Connection to the halting problem: Turing proved (1936) that no algorithm can decide whether an arbitrary programme halts. This is deeply connected to Gödel’s work — both reveal fundamental limits of formal reasoning. If we could solve the halting problem, we could decide all Gödel sentences, and vice versa.
Mathematics is not broken: Gödel’s theorems do not mean mathematics is unreliable. They mean mathematics is richer than any single formal system can capture. For any system, there are truths beyond its reach — but we can always build a stronger system that proves those truths (which will then have its own unprovable truths, and so on forever).

Where You’ll See This

Gödel’s theorems set permanent boundaries on what artificial intelligence can achieve. No matter how powerful a reasoning system becomes — whether it’s a theorem prover, a formal verification tool, or a future AGI — it cannot be both consistent and complete. This is why modern AI verification tools for safety-critical software (aircraft control systems, medical devices) must always be supplemented by human judgement. The incompleteness theorems guarantee that no automated system can catch every possible logical flaw in its own reasoning.
