Phase Plane Analysis & Nonlinear Dynamics

●●●● Advanced

Trajectories, critical points, and the geometry of dynamical systems

1. Systems of ODEs & the Phase Plane

Many phenomena in physics, biology, and engineering are governed not by a single differential equation but by a system of coupled equations. In two dimensions, we write:

dx/dt = f(x, y)
dy/dt = g(x, y)

The phase plane is the x–y plane in which we plot the trajectory (x(t), y(t)) as time evolves. Each initial condition (x₀, y₀) launches a unique trajectory. The collection of all such trajectories is the phase portrait.

A critical point (also called an equilibrium or fixed point) is a point (x₀, y₀) where both f(x₀, y₀) = 0 and g(x₀, y₀) = 0. At a critical point the system is stationary — no motion occurs. The behaviour of trajectories near a critical point reveals the system’s qualitative dynamics.

The phase plane converts a question about time evolution into a question about geometry. Instead of solving the equations analytically, we read stability, oscillation, and long-term behaviour directly from the portrait.

2. Classification of Critical Points

Linearisation & the Jacobian

Near a critical point (x₀, y₀), we linearise the system using the Jacobian matrix:

J = [ ∂f/∂x   ∂f/∂y ]
   [ ∂g/∂x   ∂g/∂y ]   evaluated at (x₀, y₀)

The eigenvalues λ₁, λ₂ of J determine the type and stability of the critical point:

TypeEigenvaluesBehaviour
Stable node λ₁, λ₂ < 0, real Trajectories approach; exponential decay
Unstable node λ₁, λ₂ > 0, real Trajectories recede; exponential growth
Saddle point λ₁ > 0, λ₂ < 0 Attract along one eigenvector, repel along the other
Stable spiral Complex, Re(λ) < 0 Oscillatory approach to equilibrium
Unstable spiral Complex, Re(λ) > 0 Oscillatory departure from equilibrium
Centre Pure imaginary ±iω Closed orbits; neutrally stable

▶ Derivation: eigenvalue classification from the characteristic equation

For a 2×2 matrix J with trace τ = a + d and determinant Δ = ad − bc, the characteristic equation is:

λ² − τλ + Δ = 0   ⇒   λ = (τ ± √(τ² − 4Δ)) / 2

• If τ² − 4Δ > 0: two distinct real eigenvalues (node or saddle)
• If τ² − 4Δ < 0: complex conjugate pair (spiral or centre)
• If τ² − 4Δ = 0: repeated eigenvalue (degenerate node or star node)
• Stability: Re(λ) < 0 iff τ < 0 (with Δ > 0)
• Saddle: Δ < 0 (eigenvalues have opposite signs)

4. Nullclines

Where the Flow Is Purely Horizontal or Vertical

A dx/dt = 0 nullcline is the set of points where f(x, y) = 0. On this curve, dx/dt = 0, so the flow is purely vertical (only dy/dt acts).

A dy/dt = 0 nullcline is the set of points where g(x, y) = 0. On this curve, dy/dt = 0, so the flow is purely horizontal (only dx/dt acts).

At any intersection of the two nullclines, both dx/dt = 0 and dy/dt = 0 simultaneously — making that point a critical point.

f(x, y) = 0  ⇒  vertical flow (dx/dt = 0)
g(x, y) = 0  ⇒  horizontal flow (dy/dt = 0)
Intersection  ⇒  critical point (no flow)

The nullclines divide the phase plane into regions. Within each region, the signs of dx/dt and dy/dt are constant, so we can determine the general direction of flow without computing any trajectory. This makes nullcline analysis a powerful qualitative tool.

Nullclines are visible on the interactive phase portrait above as coloured curves. In each region between nullclines, the direction field arrows point consistently in one quadrant direction — the signs of dx/dt and dy/dt are uniform within each region.

5. Limit Cycles

Isolated Periodic Orbits

A limit cycle is an isolated closed trajectory in the phase plane. Unlike the centres of linear systems (where every nearby orbit is also closed), a limit cycle is surrounded by trajectories that spiral either towards it (stable limit cycle) or away from it (unstable limit cycle).

The classic example is the van der Pol oscillator:

dx/dt = y
dy/dt = μ(1 − x²)y − x

For μ > 0, the origin is an unstable spiral, but trajectories cannot escape to infinity. They settle onto a stable limit cycle whose shape depends on μ. For small μ the cycle is nearly circular; for large μ it becomes a relaxation oscillation with sharp transitions.

Poincaré–Bendixson Theorem: In a two-dimensional autonomous system, if a trajectory remains in a bounded region that contains no critical points, then it must approach a closed orbit. This theorem guarantees the existence of limit cycles in many nonlinear systems — a result that has no analogue in higher dimensions.

Select the Van der Pol system in the interactive phase portrait above and increase μ. Launch trajectories from both inside and outside the limit cycle — they all converge to the same closed orbit, confirming its stability.

6. Worked Examples

Example 1: Lotka–Volterra Predator–Prey

Classify the critical points of the system:

dx/dt = x − xy   (prey)
dy/dt = −y + xy   (predator)

Step 1: Find critical points

Set x − xy = 0  ⇒  x(1 − y) = 0  ⇒  x = 0 or y = 1
Set −y + xy = 0  ⇒  y(x − 1) = 0  ⇒  y = 0 or x = 1
Critical points: (0, 0) and (1, 1)

Step 2: Compute the Jacobian

J = [ 1 − y    −x ]
    [  y      x − 1 ]

Step 3: Classify each critical point

At (0,0): J = [1, 0; 0, −1]
τ = 0, Δ = −1 < 0  ⇒  Saddle point
At (1,1): J = [0, −1; 1, 0]
τ = 0, Δ = 1 > 0, τ² − 4Δ = −4 < 0  ⇒  Centre
λ = ±i — pure imaginary, confirming closed orbits
The centre at (1, 1) means predator and prey populations oscillate periodically — they cycle endlessly without damping or growth. This is the classic Lotka–Volterra result. In real ecosystems, perturbations and nonlinear effects modify this idealized picture.

Example 2: The Simple Pendulum

Sketch the phase portrait of the undamped pendulum:

dx/dt = y
dy/dt = −sin(x)

Critical points

y = 0 and sin(x) = 0  ⇒  x = nπ, y = 0
Jacobian: J = [0, 1; −cos(x), 0]
At x = 0: Δ = cos(0) = 1 > 0, τ = 0  ⇒  Centre (pendulum at rest, hanging down)
At x = π: Δ = cos(π) = −1 < 0  ⇒  Saddle (pendulum balanced upright)

The phase portrait shows closed orbits (librations) near x = 0 for small energies, with separatrices connecting the saddle points at x = ±π. Beyond the separatrices, trajectories wrap around continuously — these correspond to the pendulum swinging over the top (rotations).
