Mean, Median, Mode & Data Visualisation

●●●● Beginner

Three measures of centre — and the art of summarising data visually

1. Introduction

Statistics is the science of collecting, organising, analysing, and interpreting data. At its heart lies a fundamental question: how can we summarise a large collection of numbers with just one or two values that capture its essence?

The three measures of central tendency each answer this question differently:

  • Mean — the arithmetic average; balances all values equally
  • Median — the middle value when data is sorted; robust against outliers
  • Mode — the most frequently occurring value; the “popular vote”

No single measure tells the whole story. A data set of salaries might have a mean pulled upward by a few very high earners, while the median gives a better sense of what a “typical” person earns. The mode tells us the most common salary bracket. Used together, these three measures reveal the shape and character of any data set.

When the mean, median, and mode are equal, the distribution is perfectly symmetrical. When they differ, the data is skewed — and the direction of skew tells us which tail is longer.

2. Mean, Median, Mode — Ungrouped Data

Definitions & Formulae

Mean (Arithmetic Average)

Mean = Σxi / n
Sum all values, divide by the count.

Median (Middle Value)

Sort the data in ascending order. Then:
If n is odd: Median = x[(n+1)/2]
If n is even: Median = (x[n/2] + x[n/2 + 1]) / 2

Mode (Most Frequent)

The value(s) that appear most often.
A data set may have one mode, multiple modes, or no mode at all.

Worked Example

Data: {4, 7, 2, 9, 4, 5, 4}
Sorted: {2, 4, 4, 4, 5, 7, 9}
Mean: (2 + 4 + 4 + 4 + 5 + 7 + 9) / 7 = 35 / 7 = 5
Median: n = 7 (odd), middle position = 4th value = 4
Mode: 4 appears 3 times (most frequent) = 4
Range: 9 − 2 = 7
Here the mean (5) is greater than the median and mode (both 4). The value 9 pulls the mean upward — a classic sign of right-skewed data.

4. Grouped Data

Working with Frequency Tables

When data is presented in class intervals (grouped data), we cannot identify exact values. Instead, we use the class mark (midpoint) of each interval as a representative value.

Mean — Three Methods

1. Direct Method

Mean = Σ(fi × xi) / Σfi
where xi = class mark, fi = frequency

2. Assumed Mean (Short-cut) Method

Mean = a + Σ(fi × di) / Σfi
where a = assumed mean, di = xi − a

3. Step Deviation Method

Mean = a + h × Σ(fi × ui) / Σfi
where ui = (xi − a) / h, h = class width

Median for Grouped Data

Median = l + [(n/2 − cf) / f] × h
l = lower boundary of median class
n = total frequency, cf = cumulative frequency before median class
f = frequency of median class, h = class width

Mode for Grouped Data

Mode = l + [(f1 − f0) / (2f1 − f0 − f2)] × h
l = lower boundary of modal class
f1 = frequency of modal class
f0 = frequency of class before modal class
f2 = frequency of class after modal class
▶ Why these formulae work

Median formula: The cumulative frequency tells us where the n/2-th observation falls. The formula linearly interpolates within the median class — it assumes values are evenly spread across the interval.

Mode formula: The modal class has the highest frequency. The formula adjusts the mode’s position within that class based on the frequencies of the neighbouring classes — pulling the mode toward whichever neighbour has a lower frequency (i.e. toward the steeper slope).

6. Data Visualisation

Types of Statistical Graphs

Bar Graph vs Histogram

A bar graph displays discrete categories with gaps between bars. A histogram displays continuous data in class intervals with no gaps between bars — because the classes are contiguous.

Histograms with Varying Class Widths

When class intervals have different widths, the y-axis must show frequency density (frequency ÷ class width), not raw frequency. Otherwise wider classes would appear disproportionately large.

Frequency Polygon

Plot the frequency (or frequency density) at each class midpoint, then connect the points with straight lines. Extend the polygon to the x-axis at both ends by adding a point at the midpoint of the hypothetical class before the first and after the last.

Ogive (Cumulative Frequency Curve)

Plot cumulative frequency against the upper class boundary. The resulting S-shaped curve lets you read off the median (at n/2), quartiles, and percentiles directly.

7. Worked Examples

Example 1 — Ungrouped Data (CBSE Style)

Question: The marks obtained by 10 students in a test are: 12, 15, 18, 12, 20, 15, 12, 18, 25, 15. Find the mean, median, and mode.

▶ Show solution
Sorted: {12, 12, 12, 15, 15, 15, 18, 18, 20, 25}
Mean: Sum = 12+12+12+15+15+15+18+18+20+25 = 162
      Mean = 162 / 10 = 16.2
Median: n = 10 (even), positions 5th and 6th
      5th = 15, 6th = 15
      Median = (15 + 15) / 2 = 15
Mode: 12 appears 3 times, 15 appears 3 times
      Bimodal: 12 and 15

Example 2 — Grouped Data (CBSE Style)

Question: The following table shows the daily wages of 50 workers. Find the mean, median, and mode.

Daily Wages (£)Workers
100–12012
120–14014
140–1608
160–1806
180–20010
▶ Show solution

Mean (Direct Method):

Class marks: 110, 130, 150, 170, 190
Σfixi = 12×110 + 14×130 + 8×150 + 6×170 + 10×190
      = 1320 + 1820 + 1200 + 1020 + 1900 = 7260
Mean = 7260 / 50 = £145.20

Median:

n/2 = 25. Cumulative frequencies: 12, 26, 34, 40, 50
Median class: 120–140 (cf reaches 26 ≥ 25)
l = 120, cf = 12, f = 14, h = 20
Median = 120 + [(25 − 12)/14] × 20
      = 120 + (13/14) × 20 = 120 + 18.57 = £138.57

Mode:

Modal class: 120–140 (highest frequency = 14)
l = 120, f1 = 14, f0 = 12, f2 = 8, h = 20
Mode = 120 + [(14 − 12)/(2×14 − 12 − 8)] × 20
      = 120 + (2/8) × 20 = 120 + 5 = £125.00
