Improper Integrals

●●●● Advanced

Extending integration beyond finite bounds and past singularities

1. Introduction

A definite integral ∫ab f(x) dx requires two things: a finite interval [a, b] and a bounded integrand on that interval. When either condition fails, we have an improper integral.

There are two types:

Type 1 — Infinite limits
One or both limits of integration extend to ±∞. For example, ∫1∞ 1/x² dx.
Type 2 — Discontinuous integrands
The integrand blows up at a point in [a, b]. For example, ∫01 1/√x dx.

In both cases the strategy is the same: replace the problematic bound with a variable, compute a proper (finite) integral, and then take a limit. If the limit is a finite number the integral converges; otherwise it diverges.

Improper integrals are not exotic curiosities — they appear throughout probability (the normal distribution requires ∫−∞∞ e−x²/2 dx), physics (gravitational potential energy), and Laplace/Fourier transforms.

2. Type 1: Infinite Limits

Definition

∫a∞ f(x) dx  =  limb→∞ ∫ab f(x) dx

If the limit exists and is finite, the integral converges. If the limit is ±∞ or does not exist, the integral diverges.

The p-test for ∫1∞ 1/xp dx

∫1∞ x−p dx  =  1/(p−1)  if p > 1   (converges)
∫1∞ x−p dx  =  ∞  if p ≤ 1   (diverges)

Worked Example 1: ∫1∞ 1/x² dx  (p = 2, converges)

Step 1: ∫1b x−2 dx = [−x−1]1b = −1/b + 1
Step 2: limb→∞ (−1/b + 1) = 0 + 1 = 1
Result: Converges to 1

Worked Example 2: ∫1∞ 1/x dx  (p = 1, diverges)

Step 1: ∫1b x−1 dx = [ln x]1b = ln(b) − 0
Step 2: limb→∞ ln(b) = ∞
Result: Diverges
The boundary between convergence and divergence at p = 1 is remarkably sharp. The function 1/x1.001 produces a finite area under the curve from 1 to ∞, but 1/x does not. This is one of the most striking facts in analysis.

3. Type 2: Discontinuous Integrands

Definition

If f(x) has a vertical asymptote at x = a (the left endpoint), we define:

∫ab f(x) dx  =  limε→0+ ∫a+εb f(x) dx

Similarly, if the singularity is at x = b (the right endpoint), approach from the left. If the singularity is at an interior point c, split the integral at c and evaluate each piece as a separate improper integral.

Worked Example: ∫01 1/√x dx

Note: f(x) = x−1/2 is unbounded as x → 0+
Step 1: ∫ε1 x−1/2 dx = [2x1/2]ε1 = 2 − 2√ε
Step 2: limε→0+ (2 − 2√ε) = 2 − 0 = 2
Result: Converges to 2
The function 1/√x shoots to infinity at x = 0, yet the area under the curve from 0 to 1 is perfectly finite. The integrand grows fast, but not fast enough — the “thinness” of the region near the asymptote compensates for the height.
▶ The p-test at a singularity

For ∫01 x−p dx with p > 0:

Converges if p < 1      Diverges if p ≥ 1

Notice this is the opposite of the p-test for infinite limits. At infinity, slower decay (smaller p) means divergence. At a singularity, faster blow-up (larger p) means divergence. The critical exponent p = 1 is the boundary in both cases.

5. The Comparison Test

Bounding Integrals by Known Results

When an antiderivative is difficult or impossible to find, we can still determine convergence by comparing against a simpler function whose behaviour is known.

Convergence (squeeze from above)
If 0 ≤ f(x) ≤ g(x) for all x ≥ a, and ∫a∞ g(x) dx converges, then ∫a∞ f(x) dx also converges.
Divergence (push from below)
If 0 ≤ g(x) ≤ f(x) for all x ≥ a, and ∫a∞ g(x) dx diverges, then ∫a∞ f(x) dx also diverges.

Worked Example: Does ∫1∞ 1/(x² + 1) dx converge?

Observe: For x ≥ 1:   0 ≤ 1/(x² + 1) ≤ 1/x²
Known: ∫1∞ 1/x² dx = 1   (converges, p = 2 > 1)
Therefore: By the Comparison Test, ∫1∞ 1/(x² + 1) dx  converges
Exact value: π/4 ≈ 0.7854  (via arctan)

Worked Example: Does ∫2∞ 1/(x − 1) dx converge?

Observe: For x ≥ 2:   1/(x − 1) ≥ 1/(2x)  (since x − 1 ≤ 2(x − 1) ≤ 2x when x ≥ 2)
Known: ∫2∞ 1/(2x) dx = ½ ∫2∞ 1/x dx = ∞  (diverges, p = 1)
Therefore: By the Comparison Test, ∫2∞ 1/(x − 1) dx  diverges
The Comparison Test is the workhorse of improper integral analysis. When you cannot find an antiderivative, compare against the p-integral family 1/xp — almost every convergence question reduces to choosing the right p.

6. Worked Examples

Problem 1: ∫1∞ e−x dx

Step 1: Write as a limit:   limb→∞ ∫1b e−x dx
Step 2: Antiderivative: ∫ e−x dx = −e−x
Step 3: Evaluate: [−e−x]1b = −e−b − (−e−1) = e−1 − e−b
Step 4: Take the limit: limb→∞ (e−1 − e−b) = e−1 − 0 = 1/e
Result: Converges to 1/e ≈ 0.3679
Exponential decay is so rapid that even starting from x = 1, the total area under e−x is just 1/e — less than one square unit. This is why exponential distributions are ubiquitous in modelling lifetimes, waiting times, and radioactive decay.

Problem 2: ∫01 ln(x) dx

Note: ln(x) → −∞ as x → 0+, so this is improper at x = 0
Step 1: Write as a limit:   limε→0+ ∫ε1 ln(x) dx
Step 2: Integration by parts: ∫ ln(x) dx = x ln(x) − x
Step 3: Evaluate: [x ln(x) − x]ε1 = (1·0 − 1) − (ε ln(ε) − ε)
Step 4: Simplify: = −1 − ε ln(ε) + ε
Step 5: Key limit: limε→0+ ε ln(ε) = 0  (L’Hôpital: ln(ε)/(1/ε) → (1/ε)/(−1/ε²) = −ε → 0)
Step 6: Therefore: limε→0+ (−1 − 0 + 0) = −1
Result: Converges to −1
Although ln(x) plunges to −∞ at x = 0, it does so slowly enough that the signed area is finite. The key step is recognising that ε ln(ε) → 0 — the linear decay of ε beats the logarithmic blow-up.
