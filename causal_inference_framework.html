Causal Inference: Potential Outcomes & DAGs

●●●● Intermediate

The fundamental problem of causal inference — and how to solve it

1. Potential Outcomes Table

Each unit has two potential outcomes: Y(0) under control and Y(1) under treatment. The individual treatment effect is Y(1) − Y(0). In an ideal experiment we observe both — but in reality we only see one. Click Hide Counterfactuals to see the fundamental problem.

Potential Outcomes

The fundamental problem of causal inference: we can never observe both Y(0) and Y(1) for the same unit at the same time. Every causal method is a strategy for filling in the missing potential outcomes.

2. Randomisation Simulator

Random assignment makes treatment independent of potential outcomes. Run many randomisations to see the sampling distribution of the ATE estimator converge to the truth. Then introduce selection bias to see what goes wrong.

3. DAG Builder

Build a directed acyclic graph. Click to add nodes, drag between nodes to add arrows. The page identifies backdoor paths from Treatment to Outcome and suggests valid adjustment sets.

DAG Controls

Backdoor criterion: A set of variables Z satisfies the backdoor criterion relative to (T, Y) if no node in Z is a descendant of T, and Z blocks every path between T and Y that contains an arrow into T.
