Partial Fractions

●●●● Intermediate

Splitting rational expressions into simpler pieces — the reverse of adding fractions

1. Introduction

Why would we want to decompose a fraction into simpler parts? Consider a familiar example from arithmetic: the fraction 7/12 can be written as 1/3 + 1/4. Neither form is “simpler” on its own — but the split version reveals the structure hiding inside the combined fraction.

Partial fraction decomposition does the same thing for algebraic fractions. Given a rational expression P(x)/Q(x) where the denominator factors, we split it into a sum of simpler fractions — each with a single factor in its denominator.

This technique is indispensable across mathematics:

  • Integration — turning a complicated rational integrand into terms we can integrate directly using logarithms and arctangent
  • Inverse Laplace transforms — splitting transfer functions into components whose inverses are known
  • Series expansions — expressing rational functions as power series by decomposing into geometric-series-friendly forms
Just as 7/12 = 1/3 + 1/4 reveals hidden structure in arithmetic, partial fractions reveal the algebraic anatomy of rational expressions. The denominator’s factorisation dictates the shape of the decomposition.

2. Linear Factors

Distinct Linear Factors

When the denominator factors into distinct linear terms, each factor gets a constant numerator. For a rational expression P(x)/[(x − a)(x − b)], we write:

P(x) / [(x − a)(x − b)]  =  A / (x − a)  +  B / (x − b)

Method: Multiply both sides by the full denominator to clear fractions, then either:

  • Substitute convenient values — set x = a to find A, set x = b to find B (the other term vanishes)
  • Compare coefficients — expand and match powers of x on both sides

The Cover-Up Rule

A quick shortcut: to find the constant above a factor (x − a), “cover up” that factor in the original expression and evaluate the remainder at x = a.

To find A: Cover up (x − a) in the original, evaluate at x = a
A = P(a) / (a − b)

Worked Example

Decompose: (3x + 5) / [(x + 1)(x − 2)]
Step 1: Set up the form: A/(x + 1) + B/(x − 2)
Step 2: Multiply through: 3x + 5 = A(x − 2) + B(x + 1)
Step 3: Set x = 2:   3(2) + 5 = A(0) + B(3)  ⇒  11 = 3B  ⇒  B = 11/3
Step 4: Set x = −1:   3(−1) + 5 = A(−3) + B(0)  ⇒  2 = −3A  ⇒  A = −2/3
Result: −2/3/(x + 1) + 11/3/(x − 2)
▶ Verify by recombining

Recombine: [−2/3(x − 2) + 11/3(x + 1)] / [(x + 1)(x − 2)]

Numerator: −2/3 · x + 4/3 + 11/3 · x + 11/3 = (9/3)x + 15/3 = 3x + 5 ✓

3. Repeated Factors

When a Factor Appears More Than Once

If a linear factor (x − a) appears with multiplicity n in the denominator, the decomposition requires one term for each power from 1 up to n:

For (x − a)²:   A/(x − a) + B/(x − a)²
For (x − a)³:   A/(x − a) + B/(x − a)² + C/(x − a)³

A common mistake is writing only A/(x − a)² and omitting the A/(x − a) term. Without all powers, the decomposition is incomplete and cannot match the original numerator.

Worked Example

Decompose: (5x + 3) / (x − 1)²
Step 1: Set up: A/(x − 1) + B/(x − 1)²
Step 2: Multiply through: 5x + 3 = A(x − 1) + B
Step 3: Set x = 1:   5(1) + 3 = A(0) + B  ⇒  B = 8
Step 4: Compare x¹ coefficients:   5 = A  ⇒  A = 5
Result: 5/(x − 1) + 8/(x − 1)²
The cover-up rule still works for the highest-power term — cover up (x − a)n and evaluate at x = a. For lower-power terms, use coefficient comparison or differentiation.

4. Irreducible Quadratic Factors

When the Denominator Has a Quadratic That Cannot Be Factored

An irreducible quadratic is a quadratic factor x² + bx + c whose discriminant b² − 4c is negative — it has no real roots and cannot be factored further over the reals.

The crucial difference: the numerator above an irreducible quadratic must be linear (Bx + C), not just a constant. This is because a quadratic factor has degree 2, and the numerator must have degree at most 1 (one less than the factor).

P(x) / [(x − a)(x² + bx + c)]  =  A / (x − a)  +  (Bx + C) / (x² + bx + c)

Worked Example

Decompose: (2x² + x + 1) / [(x − 1)(x² + 1)]
Step 1: Set up: A/(x − 1) + (Bx + C)/(x² + 1)
Step 2: Multiply through: 2x² + x + 1 = A(x² + 1) + (Bx + C)(x − 1)
Step 3: Set x = 1:   2 + 1 + 1 = A(2) + 0  ⇒  4 = 2A  ⇒  A = 2
Step 4: Compare x²:   2 = A + B = 2 + B  ⇒  B = 0
Step 5: Compare x&sup0;:   1 = A − C = 2 − C  ⇒  C = 1
Result: 2/(x − 1) + 1/(x² + 1)
If you mistakenly use just B/(x² + bx + c) instead of (Bx + C)/(x² + bx + c), the system of equations will be inconsistent — there won’t be enough free parameters to match all coefficients. A linear numerator for each irreducible quadratic factor is essential.

6. Application to Integration

Why Partial Fractions Matter for Calculus

The integral of a complicated rational function is often impossible to evaluate directly. Partial fractions transform it into a sum of integrals we know how to solve — logarithms and arctangents.

Worked Example

Evaluate: ∫ dx / (x² − 1)
Step 1 — Factor: x² − 1 = (x − 1)(x + 1)
Step 2 — Decompose: 1 / (x² − 1) = A/(x − 1) + B/(x + 1)
Step 3 — Cover-up:
  A: cover (x − 1), set x = 1:   1/(1 + 1) = 1/2
  B: cover (x + 1), set x = −1:   1/(−1 − 1) = −1/2
Step 4 — Integrate each term:
  ∫ [1/2 · 1/(x − 1) + (−1/2) · 1/(x + 1)] dx
  = ½ ln|x − 1| − ½ ln|x + 1| + C
Simplified: ½ ln|(x − 1)/(x + 1)| + C

The key integration rules at work:

∫ A/(x − a) dx = A · ln|x − a| + C
∫ A/(x − a)² dx = −A/(x − a) + C
∫ 1/(x² + k²) dx = (1/k) arctan(x/k) + C
▶ Worked example with a quadratic factor

Evaluate: ∫ (2x² + x + 1) / [(x − 1)(x² + 1)] dx

From Section 4, we found: 2/(x − 1) + 1/(x² + 1)

Therefore:

∫ 2/(x − 1) dx + ∫ 1/(x² + 1) dx
= 2 ln|x − 1| + arctan(x) + C

Without partial fractions, this integral would require far more elaborate techniques.

Every rational function (where the degree of the numerator is less than the degree of the denominator) can be integrated using partial fractions — the result is always a combination of logarithms, arctangents, and rational terms.
