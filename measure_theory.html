Measure Theory

●●●● Expert

How to measure the unmeasurable — from length to probability to integration

1. Why We Need Measure Theory

The Riemann integral cannot handle functions with “too many” discontinuities. The Dirichlet function — 1 on the rationals, 0 on the irrationals — is the classic example. Every Riemann sum oscillates between 0 and 1. But Lebesgue says: the rationals have measure zero, so the integral equals 0.

The Dirichlet Function

The Dirichlet function: f(x) = 1 if x is rational, 0 if x is irrational.
Banach–Tarski & non-measurable sets

The Banach–Tarski paradox shows that, assuming the Axiom of Choice, a solid ball can be decomposed into finitely many pieces and reassembled into two balls of the same size. The catch: the pieces are non-measurable — they have no well-defined volume. This is precisely why we need measure theory: to carefully define which sets can be measured.

5. Measure & Probability

Why probability needs measure theory

Kolmogorov’s 1933 axiomatisation of probability is built entirely on measure theory. Without σ-algebras, we cannot even define conditional probability rigorously. Without Lebesgue integration, we cannot compute expectations of general random variables. Measure theory is not just a foundation for analysis — it is the language of modern probability.

Where You'll See This

Measure theory provides the rigorous foundation for probability and integration. Without it, we can’t properly define probability on continuous spaces (like “pick a random real number”). It resolves paradoxes like the Banach-Tarski paradox and is essential for mathematical finance (the risk-neutral measure in option pricing) and quantum mechanics (probability measures on Hilbert spaces).

Try This

The Cantor set is constructed by repeatedly removing the middle third of intervals. What is its total “length” (Lebesgue measure)?

Start with [0,1] (length 1). Remove 1/3 (length 1/3). Remove 2 pieces of 1/9 (total 2/9). Total removed = 1/3 + 2/9 + 4/27 + … = (1/3)/(1 − 2/3) = 1. So the Cantor set has measure ZERO — yet it contains uncountably many points! It is one of the most striking examples of a “large” set with zero measure.
