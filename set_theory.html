Russell’s Paradox: Consider the set R of all sets that do not contain themselves. Does R contain itself? If it does, it shouldn’t; if it doesn’t, it should. This paradox forced mathematicians to develop axiomatic set theory (ZFC) — placing careful rules on which collections count as “sets” to avoid contradictions.
Why axioms matter: Naïve set theory — where any definable collection is a set — leads to paradoxes. The Zermelo–Fraenkel axioms with the Axiom of Choice (ZFC) provide a consistent foundation. Nearly all modern mathematics is built on this framework.

Where You'll See This

Every database query is set theory in disguise. SQL’s UNION, INTERSECT, and EXCEPT map directly to ∪, ∩, and set difference. When you filter rows with WHERE, you’re selecting a subset. Understanding De Morgan’s laws helps you simplify complex query conditions — and the same logic powers search engines, spam filters, and access control systems.

Try This

Using the Venn diagram above, compare (A ∪ B) ∩ C with A ∪ (B ∩ C). Are they always the same? Try different set sizes to find a case where they differ.

They are not the same! Union and intersection do not distribute symmetrically. Try A = {1,2}, B = {3}, C = {2,3}. Then (A ∪ B) ∩ C = {2,3} but A ∪ (B ∩ C) = {1,2,3}. This is why bracket placement matters — just like in arithmetic with + and ×.
