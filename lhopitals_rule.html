L’Hôpital’s Rule

●●●● Intermediate

When direct substitution fails — differentiate top and bottom to find the limit

1. Introduction — Indeterminate Forms

In calculus, we frequently encounter limits where direct substitution yields an expression like 0/0 or ∞/∞. These are called indeterminate forms — they tell us nothing about the actual value of the limit. The expression 0/0 could represent any number at all (consider that sin(x)/x → 1, but x²/x → 0, and x/x² → ∞ as x → 0).

The French mathematician Guillaume de l’Hôpital published a remarkable insight in 1696 (based on work by Johann Bernoulli): when the ratio f(x)/g(x) gives an indeterminate form, the limit of the ratio equals the limit of the ratio of their derivatives:

limx→a f(x) / g(x)  =  limx→a f′(x) / g′(x)

This works because near the point a, both functions behave like their tangent lines. The ratio of two functions that both vanish at a is determined by how fast they vanish — and the speed is captured by the derivative.

L’Hôpital’s Rule replaces the question “what does f/g approach?” with the simpler question “how do their slopes compare?” Near the indeterminate point, the functions are well-approximated by their tangent lines, so the ratio of slopes gives the ratio of values.

2. The Rule — Formal Statement

Conditions & Statement

Suppose f and g are differentiable on an open interval containing a (except possibly at a itself), and that:

1. limx→a f(x) = 0 and limx→a g(x) = 0   (the 0/0 form)
— or —
1′. limx→a |f(x)| = ∞ and limx→a |g(x)| = ∞   (the ∞/∞ form)
2. g′(x) ≠ 0 for all x near a (except possibly at a)
3. limx→a f′(x)/g′(x) exists (or equals ±∞)

Then:

limx→a f(x) / g(x)  =  limx→a f′(x) / g′(x)

The rule can be applied repeatedly — if f′/g′ is still indeterminate, differentiate again to get f″/g″, and so on, provided the conditions are met at each step.

The rule also applies when a = ±∞ (limits at infinity), with the same conditions adapted to the behaviour as x grows without bound.

▶ Proof sketch via Cauchy’s mean value theorem

Cauchy’s Mean Value Theorem: If f and g are continuous on [a, b] and differentiable on (a, b), with g′(x) ≠ 0 on (a, b), then there exists c ∈ (a, b) such that:

[f(b) − f(a)] / [g(b) − g(a)] = f′(c) / g′(c)

For the 0/0 case, set a as the point of interest (where f(a) = g(a) = 0). For any x near a, apply Cauchy’s theorem on [a, x]:

f(x) / g(x) = [f(x) − 0] / [g(x) − 0] = f′(c) / g′(c)

where c lies between a and x. As x → a, the point c is squeezed towards a as well. If limx→a f′(x)/g′(x) = L, then f′(c)/g′(c) → L, and therefore f(x)/g(x) → L. The ∞/∞ case requires a more delicate argument but follows the same spirit.

3. Worked Examples

Example 1: limx→0 sin(x) / x

Check form: sin(0)/0 = 0/0  ✓ indeterminate
Apply rule: limx→0 cos(x) / 1
Evaluate: cos(0) / 1 = 1

Example 2: limx→∞ ln(x) / x

Check form: ln(∞)/∞ = ∞/∞  ✓ indeterminate
Apply rule: limx→∞ (1/x) / 1
Evaluate: 1/∞ = 0
Meaning: the logarithm grows, but infinitely slower than x itself.

Example 3: limx→0 (ex − 1) / x

Check form: (e0 − 1)/0 = 0/0  ✓ indeterminate
Apply rule: limx→0 ex / 1
Evaluate: e0 / 1 = 1
Meaning: near x = 0, ex ≈ 1 + x, so (ex − 1) ≈ x.

Example 4: limx→0+ x · ln(x)

Form: 0 × (−∞)  not directly 0/0 or ∞/∞
Rewrite: x · ln(x) = ln(x) / (1/x)  —  now ∞/∞
Apply rule: limx→0+ (1/x) / (−1/x²)
Simplify: (1/x) × (−x²) = −x
Evaluate: limx→0+ (−x) = 0
Meaning: x shrinks to zero faster than ln(x) diverges to −∞.
Example 4 illustrates a key technique: when the limit is not in 0/0 or ∞/∞ form, rewrite it algebraically to create one of the standard indeterminate forms. The rewrite ln(x) / (1/x) converts a 0 × ∞ product into an ∞/∞ quotient.

5. Other Indeterminate Forms

Converting to 0/0 or ∞/∞

L’Hôpital’s Rule applies directly only to 0/0 and ∞/∞. However, five other indeterminate forms can be converted into these standard forms with algebraic manipulation.

FormExampleConversion Strategy
0 × ∞ x · ln(x) as x → 0+ Rewrite as f/(1/g) to get 0/0 or as g/(1/f) to get ∞/∞
∞ − ∞ 1/x − 1/sin(x) as x → 0 Combine over a common denominator to get a 0/0 quotient
00 xx as x → 0+ Take ln: ln(y) = g · ln(f), giving a 0 × ∞ form, then exponentiate
1∞ (1 + 1/x)x as x → ∞ Take ln: ln(y) = g · ln(f), giving an ∞ × 0 form, then exponentiate
∞0 x1/x as x → ∞ Take ln: ln(y) = (1/x) · ln(x), giving a 0 × ∞ form, then exponentiate
The logarithmic trick (setting y = fg, then evaluating ln(y) = g · ln(f)) is the universal tool for exponential indeterminate forms. Once you find lim ln(y) = L, the original limit is eL.

6. When L’Hôpital’s Rule Fails

Circular Reasoning

The classic example: proving limx→0 sin(x)/x = 1 using L’Hôpital’s Rule requires differentiating sin(x) to get cos(x). But the standard proof that d/dx[sin(x)] = cos(x) itself uses the fact that limx→0 sin(x)/x = 1. This creates a circular argument.

Standard derivative proof:
d/dx[sin(x)] = limh→0 [sin(x+h) − sin(x)] / h
  = limh→0 [sin(x)cos(h) + cos(x)sin(h) − sin(x)] / h
  = sin(x) · limh→0 (cos(h)−1)/h + cos(x) · limh→0 sin(h)/h
The red term is the very limit we are trying to prove!

This is why lim sin(x)/x = 1 must be established independently (typically via the squeeze theorem using geometric arguments about arc length and area) before L’Hôpital’s Rule can be used on other problems involving trigonometric derivatives.

When the Derivative Ratio Has No Limit

L’Hôpital’s Rule requires that lim f′/g′ exists (condition 3 above). If that limit does not exist, the rule gives no information — but the original limit may still exist.

Consider: limx→∞ (x + sin(x)) / x
Direct: = 1 + sin(x)/x → 1   (since sin(x)/x → 0)
L’Hôpital: limx→∞ (1 + cos(x)) / 1 = 1 + cos(x)
This oscillates between 0 and 2 — no limit exists!

The failure of L’Hôpital’s Rule here does not mean the original limit fails to exist. It simply means the rule is not the right tool for this particular problem. Always check whether simpler methods (direct simplification, factoring, the squeeze theorem) can resolve the limit first.

L’Hôpital’s Rule is an implication, not an equivalence. “If the derivative ratio has a limit, then so does the original” is true. The converse — “if the original has a limit, then the derivative ratio does” — is false.
