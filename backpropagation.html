Backpropagation

How blame flows backwards — the chain rule as the engine of learning

A neural network makes a prediction, measures its error, then asks: which weight was most responsible? Backpropagation answers by sending the gradient of the loss backwards through the network — one layer at a time — using nothing more than the chain rule. It is the single algorithm that makes deep learning possible.

FORWARD: x → z = Wx+b → a = ReLU(z) → ŷ → L = loss(ŷ, y)
BACKWARD: ∂L/∂ŷ → ∂L/∂z → ∂L/∂W
CHAIN RULE: ∂L/∂w = ∂L/∂a × ∂a/∂z × ∂z/∂w
Each factor is a local derivative. Backprop multiplies them together along the path from output to weight — this is the chain rule.
Backprop is just the chain rule — applied systematically to a graph of computations. Rumelhart, Hinton and Williams published it in 1986. It took another 25 years for it to change the world.

In matrix form, backprop computes vector–Jacobian products (vᵀJ) efficiently, layer by layer. This is why backprop is O(n) in network depth rather than O(n²) — it never explicitly constructs the full Jacobian matrix. The computational cost of the backward pass is approximately equal to the forward pass.

Every gradient in a neural network is computed by the chain rule, flowing backwards from loss to input.

Sigmoid: σ′(x) = σ(x)(1 − σ(x)) ≤ 0.25 → gradients shrink each layer
ReLU: f′(x) = 1 if x > 0, else 0 → gradients pass through unchanged
BatchNorm: normalises activations → stabilises gradient magnitudes
Before ReLU and residual connections, training deep networks was nearly impossible — gradients would vanish before reaching early layers. The “deep learning revolution” required solving this exact problem.
Residual connections provide a direct gradient highway through the network. In a residual block, output = input + f(input), so ∂output/∂input = 1 + ∂f/∂input. Even if ∂f/∂input vanishes, the gradient still flows through the identity path. This is why modern transformers with 100+ layers can train stably — and why the Transformer Architecture uses residual connections around every attention and FFN block.

Sigmoid kills gradients. ReLU preserves them. Batch normalisation tames them. This is why depth became possible.

Why This Matters

Every parameter in every neural network on Earth was tuned by backpropagation. When GPT-4 learns to reason, when AlphaFold predicts a protein’s shape, when a self-driving car recognises a pedestrian — the chain rule, applied backwards through millions of operations, is the engine that made it happen.

Backpropagation reduces the cost of computing all gradients from O(n²) to O(n) — the same cost as a single forward pass. Without this efficiency, deep learning would be computationally impossible.

References: Rumelhart, Hinton & Williams (1986), “Learning representations by back-propagating errors” · Hochreiter (1991), vanishing gradient problem · He et al. (2015), deep residual learning.
