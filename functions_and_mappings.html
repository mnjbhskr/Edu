Functions & Mappings

●●●● Intermediate

How inputs become outputs — and the algebra of combining, reversing, and transforming functions

1. What Is a Function?

Definition & Notation

A function is a mapping that assigns exactly one output to every valid input. We write f(x) to mean “the output of function f when the input is x.” This notation, introduced by Euler, is one of the most powerful shorthands in mathematics.

Not every relation between inputs and outputs qualifies as a function. A relation like x² + y² = 25 (a circle) gives two y-values for most x-values — so it is not a function from x to y.

Types of Mapping

One-to-one — each output comes from exactly one input. Example: f(x) = 2x + 3
Many-to-one — different inputs can produce the same output. Example: f(x) = x²
One-to-many — an input maps to multiple outputs. This is NOT a function.

The Vertical Line Test

Draw any vertical line on the graph. If it crosses the curve at more than one point, the relation is not a function. This is because a vertical line represents a single x-value, and a function demands only one y-value for each x.

A function is essentially a machine: feed it an input, and it produces exactly one output. The vertical line test is the graphical embodiment of this rule — if you can ever get two outputs for one input, the machine is broken.

2. Domain & Range

Valid Inputs and Possible Outputs

The domain of a function is the set of all valid inputs. The range is the set of all outputs the function actually produces. Finding these requires understanding what can go wrong.

Natural Domain Restrictions

Division by zero: f(x) = 1/(x − 2) → domain: x ≠ 2
Square root of negative: g(x) = √(x − 4) → domain: x ≥ 4
Logarithm of non-positive: h(x) = ln(x) → domain: x > 0

Worked Examples

f(x) = 1/(x − 2)
Domain: x ∈ ℜ, x ≠ 2   |   Range: y ∈ ℜ, y ≠ 0
g(x) = √(x − 4)
Domain: x ≥ 4   |   Range: y ≥ 0

The domain tells you where the function lives; the range tells you what values it can reach. Always check for division by zero, square roots of negatives, and logarithms of non-positives when determining the natural domain.

3. Composite Functions

Combining Functions

The composite function fg(x) means “apply g first, then apply f to the result.” Read it right to left: g acts on x, then f acts on g(x).

fg(x) = f(g(x))  —  apply g first, then f

Order matters. In general, fg(x) ≠ gf(x). Composition is not commutative.

Worked Example

Let f(x) = 2x + 1,   g(x) = x²
fg(x) = f(g(x)) = f(x²) = 2(x²) + 1 = 2x² + 1
gf(x) = g(f(x)) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1
fg(3) = 2(9) + 1 = 19   |   gf(3) = (7)² = 49    — different!

Composition is the algebraic equivalent of a pipeline: data flows through one function into the next. Reversing the pipeline (swapping the order) usually changes the outcome — just as “wash then dry” differs from “dry then wash.”

4. Inverse Functions

Undoing a Function

The inverse function f ¹(x) reverses the effect of f. If f takes a to b, then f ¹ takes b back to a. An inverse exists only when f is one-to-one — every output comes from exactly one input.

Finding the Inverse Algebraically

1. Write y = f(x)
2. Swap x and y
3. Solve for y → this is f−1(x)

Worked Example

Given: f(x) = 2x + 3
Step 1: y = 2x + 3
Step 2: x = 2y + 3
Step 3: y = (x − 3)/2
Therefore: f−1(x) = (x − 3)/2
Check: f(f−1(x)) = 2·(x − 3)/2 + 3 = x − 3 + 3 = x   ✓

Graphically, f−1 is the reflection of f in the line y = x. Every point (a, b) on f maps to (b, a) on f−1.

The reflection property is not a coincidence. Swapping x and y in the equation is the algebraic version of reflecting in y = x — both operations exchange the roles of input and output.

5. Graph Transformations

The Six Key Transformations

Every A-level student must master these six transformations. They modify the graph of y = f(x) in predictable ways. The tricky part: transformations inside the brackets (affecting x) work in the opposite direction to what you might expect.

y = f(x) + a — vertical shift UP by a
y = f(x + a) — horizontal shift LEFT by a (opposite!)
y = a·f(x) — vertical stretch, scale factor a
y = f(a·x) — horizontal stretch, scale factor 1/a (reciprocal!)
y = −f(x) — reflection in the x-axis
y = f(−x) — reflection in the y-axis

Why does f(x + a) shift left? Think of it this way: to get the same output that f used to give at x = 5, you now need x = 5 − a. Every point “arrives earlier” by a units, which means the whole graph moves left by a.

6. The Modulus Function

Absolute Value Graphs

The modulus (absolute value) function |x| returns the magnitude of x — always non-negative. It produces two important graph transformations:

y = |f(x)| — any part of f(x) below the x-axis is reflected above it
y = f(|x|) — the graph for x ≥ 0 is reflected to the left (replacing the left side)

Solving Modulus Equations

To solve |f(x)| = k (where k ≥ 0), solve both f(x) = k and f(x) = −k. For modulus inequalities, sketch the graph — the solution is where the graph lies in the required region.

|f(x)| and f(|x|) look similar but do very different things. The first guarantees every output is non-negative (reflecting below the x-axis up). The second guarantees every input is non-negative (reflecting the right half to the left, creating symmetry about the y-axis).

7. Worked Examples

A-Level Style Problems

▶ Example 1: Find fg(x) and state its domain
Given: f(x) = 1/(x − 1),   g(x) = √x
Step 1: Form fg(x)
fg(x) = f(g(x)) = f(√x) = 1/(√x − 1)
Step 2: Domain of g
g(x) = √x requires x ≥ 0
Step 3: Domain restriction from f
f needs its input ≠ 1, so √x ≠ 1, i.e. x ≠ 1
Result: fg(x) = 1/(√x − 1),   domain: x ≥ 0, x ≠ 1
▶ Example 2: Find f−1 and verify ff−1(x) = x
Given: f(x) = (3x − 5)/2
Step 1: Let y = (3x − 5)/2
Step 2: Swap: x = (3y − 5)/2
Step 3: 2x = 3y − 5  ⇒  3y = 2x + 5  ⇒  y = (2x + 5)/3
Therefore: f−1(x) = (2x + 5)/3
Verification:
ff−1(x) = f((2x + 5)/3) = (3·(2x + 5)/3 − 5)/2
= (2x + 5 − 5)/2 = 2x/2 = x   ✓
▶ Example 3: Describe the transformation from y = x² to y = 3(x − 2)² + 1
Start: y = x²
Step 1: Replace x with (x − 2):   y = (x − 2)²
→ Horizontal translation 2 units to the right
Step 2: Multiply by 3:   y = 3(x − 2)²
→ Vertical stretch, scale factor 3
Step 3: Add 1:   y = 3(x − 2)² + 1
→ Vertical translation 1 unit upwards
Summary: Translate right 2, stretch vertically ×3, then translate up 1.
Vertex moves from (0, 0) to (2, 1).
