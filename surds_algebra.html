Surds & Algebraic Manipulation

●●●● Beginner

Exact values, simplified roots, and the art of tidy fractions

1. What Are Surds?

Irrational Roots in Exact Form

A surd is an irrational root that cannot be simplified to a whole number or fraction. Numbers like √2, √3, and √5 are surds — their decimal expansions go on forever without repeating.

We keep surds in exact form because decimal approximations lose precision. Writing √2 rather than 1.41421… means our answer is perfectly accurate — no rounding error, no matter how many subsequent calculations we perform.

Which roots are surds?

Surd: √2 = 1.41421356…  (irrational — never terminates)
Surd: √3 = 1.73205080…
Surd: ∛5 = 1.70997594…
Not a surd: √4 = 2  (rational — simplifies to a whole number)
Not a surd: √(9/16) = 3/4  (rational)

Simplifying surds

To simplify a surd, find the largest perfect square factor of the number under the root sign:

Example: √12 = √(4 × 3) = √4 × √3 = 2√3
Example: √72 = √(36 × 2) = √36 × √2 = 6√2
Example: √50 = √(25 × 2) = √25 × √2 = 5√2
Example: √200 = √(100 × 2) = √100 × √2 = 10√2
Look for the largest perfect square factor: 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144… The bigger the factor you spot first, the fewer steps you need. If you miss the largest, you can always simplify further — √72 = √(4×18) = 2√18 = 2√(9×2) = 6√2.

2. Rules of Surds

The Three Laws

Multiplication Rule
√(ab) = √a × √b
Example: √(3 × 5) = √3 × √5 = √15
Example: √2 × √8 = √16 = 4
Division Rule
√(a/b) = √a / √b
Example: √(12/3) = √12 / √3 = 2√3 / √3 = 2
Example: √(50/2) = √50 / √2 = √25 = 5
Addition of Like Surds
a√n + b√n = (a + b)√n
Example: 3√5 + 7√5 = 10√5
Example: 5√2 − 2√2 = 3√2
Common mistake: √(a + b) ≠ √a + √b.
For example: √(9 + 16) = √25 = 5, but √9 + √16 = 3 + 4 = 7. These are not equal.
You can only add surds when the number under the root sign is the same — “like surds”.
▶ Why does √(a + b) ≠ √a + √b?

The square root function is concave — geometrically, its graph curves downward. By Jensen’s inequality, for any concave function f and positive values a, b:

f(a + b) ≤ f(a) + f(b)

Equality holds only when a = 0 or b = 0. In every other case, √(a + b) is strictly less than √a + √b. This is why the “rule” does not work — it would violate a fundamental property of the function.

Multiplying brackets with surds

Expand: (2 + √3)(4 − √3)
Step 1: = 2 × 4 + 2 × (−√3) + √3 × 4 + √3 × (−√3)
Step 2: = 8 − 2√3 + 4√3 − 3
Step 3: = 5 + 2√3
Key identity: (a + √b)(a − √b) = a² − b
Example: (3 + √5)(3 − √5) = 9 − 5 = 4
The surd disappears! This is the conjugate — the key to rationalisation.

3. Rationalising the Denominator

Removing Surds from the Bottom

Convention requires that we do not leave surds in the denominator of a fraction. Rationalising means multiplying top and bottom by a suitable expression to eliminate the surd below.

Type 1: Single surd in denominator

Rationalise: 1 / √a
Multiply by: √a / √a
Result: √a / a
Example: 5 / √3 = (5 × √3) / (√3 × √3) = 5√3 / 3

Type 2: Binomial with a surd — use the conjugate

Rationalise: 1 / (a + √b)
Multiply by: (a − √b) / (a − √b)
Result: (a − √b) / (a² − b)
Example: 3 / (2 + √5)
Step 1: = 3(2 − √5) / [(2 + √5)(2 − √5)]
Step 2: = 3(2 − √5) / (4 − 5)
Step 3: = 3(2 − √5) / (−1)
Step 4: = −6 + 3√5  or  3√5 − 6

Type 3: Two surds in denominator

Rationalise: 1 / (√a + √b)
Multiply by: (√a − √b) / (√a − √b)
Result: (√a − √b) / (a − b)
Example: 4 / (√7 − √3)
Step 1: = 4(√7 + √3) / [(√7 − √3)(√7 + √3)]
Step 2: = 4(√7 + √3) / (7 − 3)
Step 3: = 4(√7 + √3) / 4
Step 4: = √7 + √3
The conjugate trick works because of the difference of two squares: (x + y)(x − y) = x² − y². When x or y is a square root, the squaring removes the root. This is the single most important algebraic technique for handling surds in denominators.

4. Algebraic Fractions

Operations on Fractions with Variables

Algebraic fractions follow exactly the same rules as numerical fractions: find a common denominator to add or subtract, multiply straight across, flip and multiply to divide, and factorise to simplify.

Adding & subtracting

Simplify: 2/x + 3/(x+1)
Step 1: Common denominator = x(x + 1)
Step 2: = 2(x + 1) / [x(x + 1)] + 3x / [x(x + 1)]
Step 3: = [2(x + 1) + 3x] / [x(x + 1)]
Step 4: = [2x + 2 + 3x] / [x(x + 1)]
Step 5: = (5x + 2) / [x(x + 1)]

Multiplying

Simplify: [(x + 2) / (x − 3)] × [(x − 3) / (x + 5)]
Step 1: Cancel (x − 3) top and bottom
Step 2: = (x + 2) / (x + 5)

Dividing — flip and multiply

Simplify: [3x / (x + 1)] ÷ [6x² / (x + 1)²]
Step 1: = [3x / (x + 1)] × [(x + 1)² / 6x²]
Step 2: = [3x(x + 1)²] / [6x²(x + 1)]
Step 3: Cancel x and (x + 1)
Step 4: = (x + 1) / 2x

Simplifying by factorising

Simplify: (x² − 4) / (x² + 5x + 6)
Step 1: Factorise numerator: x² − 4 = (x + 2)(x − 2)
Step 2: Factorise denominator: x² + 5x + 6 = (x + 2)(x + 3)
Step 3: Cancel (x + 2)
Step 4: = (x − 2) / (x + 3)
Always factorise before multiplying or cancelling. Look for: common factors, difference of two squares (a² − b²), trinomial factorisation, and grouping. The examiner expects the simplest possible form.

6. Worked Examples — A-Level Problems

Problem 1

Express (3 + √5)² in the form a + b√5, where a and b are integers.

▶ Show solution
Expand: (3 + √5)² = (3 + √5)(3 + √5)
Step 1: = 3² + 2 × 3 × √5 + (√5)²
Step 2: = 9 + 6√5 + 5
Step 3: = 14 + 6√5
Answer: a = 14, b = 6

Problem 2

Simplify (2√3 + √6) / √3, giving your answer in the form a + √b.

▶ Show solution
Split: (2√3 + √6) / √3 = 2√3 / √3 + √6 / √3
Step 1: 2√3 / √3 = 2
Step 2: √6 / √3 = √(6/3) = √2
Step 3: = 2 + √2
Answer: a = 2, b = 2

Problem 3

Show that 1/(3 − √7) + 1/(3 + √7) = 3.

▶ Show solution
Rationalise each fraction:
First: 1/(3 − √7) × (3 + √7)/(3 + √7) = (3 + √7) / (9 − 7) = (3 + √7) / 2
Second: 1/(3 + √7) × (3 − √7)/(3 − √7) = (3 − √7) / (9 − 7) = (3 − √7) / 2
Add: (3 + √7) / 2 + (3 − √7) / 2
Step 1: = [(3 + √7) + (3 − √7)] / 2
Step 2: = 6 / 2
Step 3: = 3   ■
