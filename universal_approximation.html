Universal Approximation

Why neural networks can learn any function

A single hidden layer of sigmoid neurons can approximate any continuous function to arbitrary precision. This remarkable result — the universal approximation theorem — is the theoretical foundation of deep learning. Build up the intuition from one neuron to many, and watch the approximation emerge.

Key idea: A single sigmoid neuron outputs σ(wx + b) = 1 / (1 + e−(wx+b)). As |w| grows large, the sigmoid approaches a step function — a building block for more complex shapes.

By subtracting two shifted sigmoids we create a bump. This is the fundamental primitive — any function can be built from bumps.

The trick: h · [σ(w(x − a)) − σ(w(x − b))] produces a rectangular bump between a and b with height h, when w is large enough.

5. What Have We Learnt?

The Universal Approximation Theorem

In 1989, George Cybenko proved that a feedforward network with a single hidden layer of sigmoid neurons can approximate any continuous function on a compact set to arbitrary precision, given enough neurons. Kurt Hornik extended this in 1991, showing the result holds for any non-constant, bounded, continuous activation function — it is not special to the sigmoid.

The theorem tells us existence, not efficiency. It guarantees that a wide-enough single layer can represent any function, but says nothing about how many neurons you need or how to find the right weights. In practice, deep networks (many narrow layers) often generalise better and need fewer parameters than the astronomically wide shallow networks the theorem permits.

f(x) = Σi vi · σ(wix + bi) + c

Each sigmoid creates a step-like feature. Weighted sums of these features build bumps. Enough bumps approximate any shape. This is strikingly similar to how Fourier series build functions from sine waves — a different basis, the same principle.

Connections
  • Fourier analysis — approximation by summing basis functions is a universal theme in mathematics
  • Stone–Weierstrass theorem — the classical result that polynomials are dense in C[a,b]; universal approximation is the neural-network analogue
  • Gradient descent — the theorem says good weights exist; gradient descent is how we find them
  • Depth vs width — Telgarsky (2016) showed functions that require exponentially many neurons in a shallow network but only polynomially many in a deep one
